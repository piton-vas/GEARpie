# Бедро 180 Н·м — трёхступенчатый планетарный редуктор: итоговые варианты

Выход **180 Н·м**, 3 ступени, **i = 7.0 × 3.75 × 3.0 = 78.75** (под мотор ~3.3 Н·м). Все варианты ниже
**проходят прочность** (VDI 2736: SF ≥ 1.7, SH ≥ 1.3, Ka = 1.3, grease, 1e7 циклов) **и условие зуба b/m ≤ 12**.
Отобраны по критерию **минимум работы и материала** (меньше деталей, подшипников, фрезеровки).

## Лучшие варианты

«Подш.» = число саттелитов (на каждый нужен подшипник). «Фрез./печать шест.» = сколько зубчатых деталей
фрезеровать / печатать (3 венца + 3 солнца + саттелиты). Материал: капролон = PA6_CAST (фреза), печать = PA6 FFF + отжиг.

> детали (максимум работы) — берут, только если критичен размер.

## Как выбрать

- **Баланс работа/материал → ① CAP-PAA 3/5/5 (★).** Тяжёлые венцы печатаем (их фрезеровать дольше всего),
  фрезеруем только мелкие солнца/саттелиты из капролона, саттелитов немного (13 подшипников), Ø умеренный.
- **Без ЧПУ вообще (только печать) → ② PRINT 3/5/8** (компактный, но 16 подшипников, 8 на ст.3 — зазор 1.9 мм,
  собирать туго) или **③ PRINT 3/3/3** (9 подшипников, свободно, но Ø242 — много филамента).
- **Минимальный размер → ④ CAP-ALL 3/5/6** (Ø168, контакт с запасом SH 1.56), ценой фрезеровки и венца.

## Почему именно так (кратко)

- **Лимитирует контакт, не изгиб.** Решает σHlim и жёсткость материала.
- **Капролон под grease — лучший материал** (S-N, σHlim ≈ 63): компактнее POM-C (35) и печатного PA6 (25).
- **Печатный венец = потолок SH ≈ 1.30–1.32** и Ø не меньше ~168; больше контактного запаса (SH 1.5+) — только
  фрезерованный капролоновый венец (④).
- **Отброшены:** алюминий и POM-C на шестернях (жёсткие → задирают контакт, Ø188, алюминий ещё и тяжёлый);
  печать без отжига (слаба). Реальный выигрыш по размеру дала бы только сталь — её нет в доступных материалах.
- **Меньше саттелитов = меньше подшипников/деталей, но больше Ø:** каждые −4 подшипника ≈ +20…36 мм Ø.
  Диаметр меняется ступеньками 8 → {6,5} → 4 (между 6 и 5 один Ø ⇒ из них рационален 5).

## Оговорки

- SH печатно-венцовых (①②③⑤) у порога 1.30 — физический потолок печатного пластика по контакту, не ошибка.
- Износ оценочный (VDI 2736-2): ресурс ~5·10⁷…1.3·10⁸ циклов. При консистентной смазке + полиамид на низком PV
  износ может не снижаться — проверить испытанием либо рассмотреть масло.
- Передаточное 78.75 — под мотор ~3.3 Н·м; под другой момент пересчитать (выводы по материалам сохранятся).
