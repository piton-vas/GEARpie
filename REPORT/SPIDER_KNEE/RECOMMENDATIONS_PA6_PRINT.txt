  Мотор:      T_in = 3.7 Н·м, n_in = 150 об/мин
  Сустав:     T_out >= 60 Н·м (статически)
  Передача:   i > 32 (несимметричная двухступенчатая)
  Ступень 2:  4 сателлита (обязательно)
  Габарит:    Ø <= 200 мм
  Осевая:     желательно <= 40 мм (очень компактно)
  Материалы:  PA6_PRINT везде

ПРОБЛЕМА
-----------------------------------------------------------------------------------------------------
PA6_PRINT — мягкий материал (σHlim = 25 МПа, σFlim = 16 МПа). Для пластиковых коробок это
означает, что запас по контактной прочности (SH) критичен. Норматив VDI 2736 требует:
  SF_min >= 2.0  (по изгибу корня)
  SH_min >= 1.4  (по контактной прочности)

При компактной коробке (узкая ширина) эти требования не достижимы за счёт низких σH.

=====================================================================================================
ТЕХНОЛОГИЧЕСКИЕ ЗАМЕЧАНИЯ ДЛЯ FFF-ПЕЧАТИ PA6
=====================================================================================================

1. ОРИЕНТАЦИЯ ПЕЧАТИ
   - Зубья (фланк) должны быть ПАРАЛЛЕЛЬНЫ оси (вдоль слоёв)
   - НЕ печатать зубья поперёк слоёв (будут хрупкими)

2. ТОЛЩИНА СТЕНОК
   - Минимум 2-3 мм для венцов
   - Рёбра жесткости с углом 45° для поддержки при печати

3. ДОПУСКИ
   - Модуль, расстояния между центрами: ±0.1 мм
   - Диаметр отверстий под подшипники/валы: +0.1/-0.0 мм (вал должен войти свободно)

4. ОТЖИГ (HEAT TREAT)
   - Без отжига: параметры материала как в расчёте
   - С отжигом при 70-80°C: σFlim и σHlim повышаются на ~15%
   - Рекомендуется отжиг после печати

5. СМАЗКА
   - PA6_PRINT + PA6_PRINT: используйте сухую PTFE смазку или воск
   - НЕ использовать масла / силиконовую смазку (разъедает пластик)
   - Воск (машинный воск) хорошо работает с PA

6. ТЕПЛОВОЕ РАСШИРЕНИЕ
   - PA6 при нагреве расширяется: α ≈ 80×10⁻⁶ /°C
   - При Δt = 20°C размер изменится на 0.16% (критично для зубьев)
   - Работайте при постоянной температуре или используйте графики компенсации

=====================================================================================================
ДИАГНОСТИКА И ОТЛАДКА
=====================================================================================================

🔴 ПРИЗНАКИ НЕПРАВИЛЬНОЙ РАБОТЫ:

1. Шум, вибрация, заедание:
   - Проверьте зазор между саттелитами (не менее 0.5 мм)
   - Проверьте центровку водила (допуск радиального биения < 0.2 мм)

2. Быстрый износ зубьев:
   - Может быть вибрация или неправильная нагрузка
   - Проверьте балансировку ротора

3. Деформация венца / саттелитов:
   - Перегрев (PA6 размягчается выше 80°C)
   - Проверьте смазку, может быть трение
