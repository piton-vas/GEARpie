**Назначение:** тазобедренный сустав шагающего робота (бедро), T_out = 180 Н·м.

**Расчёт:** скил `planetary-gearbox`, VDI 2736 (пара солнце–саттелит) + Lewis/Hertz (пара саттелит–венец).

## 1. Архитектура (общая для кампании)

Кампания V1 (`REPORT/THIGH_PA6A_82_345.md`) показала: лимитирует **контакт** выходной
ступени, изгиб — почти никогда. Отсюда принципы V2:

1. **Минимальное i₃ = 3.0** (при z_сат ≥ 18): Ø венца ≈ d_солнца·(i₃−1) → большое солнце
   при маленьком венце; высокое i — на входе, где момент всего 2.3 Н·м.
2. **Максимум саттелитов на выходе:** SH ∝ d_солнца·√(n·b) — каждый добавленный саттелит
   позволяет убрать ширину или диаметр. Сумма зубьев (z_s + z_r) подбирается кратной
   большому n при достаточном зазоре вершин соседних саттелитов (≥ 1.5 мм).
3. **Ступень 1 ужата** (m 1.25 → 1.0, Ø венца 147 → 120 мм).

Передаточное: **i = 7.0 × 3.75 × 3.0 = 78.75** (все варианты V2); n_out = 1.905 об/мин;
требование к мотору 3.14 Н·м ≤ 3.3 (M3v3 ✓). Моменты по ступеням (номин., без Ka):
2.286 → 16.0 → 60.0 → 180.0 Н·м.

### 2.1. Выводы по материалам

### 2.2. Уроки гибридов печать/фрезеровка (раунд 2)

**Менять материал можно только парами.** Контакт считается по паре: жёсткий партнёр
поднимает приведённый модуль E* и σH для обоих, а держать должен слабейший (SH_pess по
меньшему σHlim):

- **K (фрезеровано только солнце ст.3, POM_C):** печатный саттелит против жёсткого
  солнца — SH падает 1.523 → **1.358 < 1.4, НЕ проходит**. Подтверждает вывод V1.
- **L (солнце+саттелиты Д16Т, венец печатный):** внешняя пара Al–Al в порядке, но
  внутренняя пара «Al-саттелит — печатный венец» убивает венец: SH = **1.259, НЕ проходит**
  (нужно b₃ ≈ 60+, теряется весь смысл).
- **I (солнце+саттелиты POM_C, венец печатный) — правильный гибрид:** обе пары согласованы
  (внешняя POM-C/POM-C; внутренняя POM-C/PA6 — мягкий венец только помогает). Фрезеруются
  лишь мелкие круглые детали (солнца Ø≤73, саттелиты Ø≤38 — токарка/фреза из прутка POM),
  а сложные внутренние зубья венцов — печатаются. Ø152 × L159, SF 3.62, SH 1.482 ✅.

## 5. Производственные примечания

- **PA6_ANNEAL:** печать FFF + отжиг обязательны (без отжига — вариант C с +8 мм ширины).
  Ось колеса при печати вертикально; заполнение несущих колёс ≥ 50–80 % или сплошное.
- Боковой зазор: профили теоретические (x = 0); для печати закладывать по толщине зуба
  ≈ 0.1–0.2 мм по месту.
- **H, ст.3:** зазор вершин соседних саттелитов всего **1.7 мм** (лимит 1.5) — следить за
  допусками печати саттелитов (da = 38.5) и позиций осей водила. Если страшно — вариант I
  (фрезерованные саттелиты точнее печатных) или J (зазор 6.5 мм).
- **Подшипники планет:** H/I — 16 опор (3+5+8), J — 15 (3+5+7); V1-финал имел 12, но при
  Ø169.5. Компактность куплена числом опор.
- **I:** солнца (Ø до 73 мм) и саттелиты (Ø до 38 мм) — токарка/фреза из прутка POM-C;
  венцы с внутренними зубьями — печать. Не менять материал «половинке» пары (уроки K/L).
