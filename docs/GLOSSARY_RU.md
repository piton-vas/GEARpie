# Глоссарий терминов GEARpie (EN → RU)

Терминология приведена в соответствие с ГОСТ 16530-83 (передачи зубчатые. термины и определения), ГОСТ 21354-87 (расчёт на прочность), DIN 3990 / VDI 2736 (оригинальные названия стандартов не переводятся).
